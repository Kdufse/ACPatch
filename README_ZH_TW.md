## 🚀🚀🚀 最新消息：核心功能優化

> **🎯🎯 智慧模組管理** - 全自動 KPM 載入機制，無需嵌入 Boot，提供更穩定的 Root 體驗

### 🔧🔧 模組系統重構
- **APM 批次刷入系統：** 支援類似 Magisk 的模組系統，支援批次刷入，更加快速高效
- **KPM 自動化載入：** 支援將任何程式碼注入核心的模組，全自動載入機制，無需手動嵌入
- **模組完整備份：** 一鍵備份所有模組，讓您安心體驗 Root，安全無憂

### 📐📐 介面設計優化
- **自訂桌布系統：** 更好的自訂系統，支援自訂桌布，打造專屬個人化介面
- **現代化 UI 設計：** 精心優化的介面設計，提供流暢的視覺體驗
- **多語言支援：** 多種特色語言，隨心切換

### 🛠🛠🛠 功能特性增強
- **線上模組下載：** 整合線上模組下載功能，快速取得熱門模組
- **全域模組排除：** 模組全域排除功能，操作更加快速便捷
- **移除自動更新：** 移除自動更新功能，提供更穩定的使用者體驗

- [x] 基於核心的 Android 裝置 Root 解決方案

## 🚀🚀🚀 下載安裝

### 📦📦 安裝

1. **下載安裝：**
   從 [發佈頁面](https://github.com/Kdufse/ACPatch/releases/latest) 下載最新的 APK 檔案

2. **安裝應用：**
   直接安裝 APK 檔案到您的 Android 裝置

3. **開始使用：**
   開啟 ACPatch 應用，按照指引完成 Root 設定

### 📱📱 系統要求

- **架構：** 支援 ARM64 架構
- **核心版本：** 支援 Android 核心版本 3.18 - 6.12

## 🙏🙏 開源致謝

本專案基於以下開源專案：

- [KernelPatch](https://github.com/bmax121/KernelPatch/) - 核心元件
- [Magisk](https://github.com/topjohnwu/Magisk) - magiskboot 和 magiskpolicy
- [KernelSU](https://github.com/tiann/KernelSU) - 應用 UI 和類似 Magisk 的模組支援
- [Sukisu-Ultra](https://github.com/SukiSU-Ultra/SukiSU-Ultra) - 參考部分介面設計
- [APatch](https://github.com/bmax121/APatch) - 超級用戶
- [FolkPatch](https://github.com/matsuzaka-yuki/FolkPatch) - 上游分支
